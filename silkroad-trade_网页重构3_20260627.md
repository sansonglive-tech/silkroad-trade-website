# 丝路山海通网页 — 第三次重构（2026-06-27 10:54）

## 目标
用户反馈上一版"太暗、太紧凑、版面间色差太大、缺少图片、logo没设计感"，要求参考山海通（shanhaimap.com）的轮播样式，做出立体的、有设计感的版本。

## 核心改动

### 1. 整体色调 → 明亮暖白系
- 背景从深色墨褐改为 **宣纸白（#FCFAF6）** + **米黄（#F2EBE0）**
- 保留朱红（#C44536）和赤金（#C8923E）作为点缀色，不占大面积
- 深色区仅保留 stats-banner（深褐色渐变）
- 区块过渡不再用突兀的纯白/纯灰，改用渐变衔接

### 2. 轮播图重做 → 立体感 + 卡片化
- 采用 **左右布局**：左文右图（不像之前全屏文字），视觉层次清晰
- 右侧用 **3D Mockup Card**（带 `perspective` 透视旋转 + hover 立体抬起效果）
- 每张卡片有 icon、标题、描述、tag 标签，感觉像一张"业务名片"
- 底部 dots + 左右箭头 + 计数器，交互完整
- 自动播放 4.5 秒

### 3. Logo 设计感提升
- 用 **SVG 自制 logo**：红底方块 + 金色波浪线（丝路）+ 金色山形 + 金色海浪 + 书法字「丝」
- 不再是简单的文字，而是一个有寓意的品牌标识
- 丝路 = 波浪线，山海 = 山形 + 海浪

### 4. 各板块配图（emoji + 色块背景）
- 每个服务卡片顶部加了 **100px 高的色块背景 + emoji 图标**
- 色块颜色按业务区分：注册→红、财税→金、签证→绿、产品→棕、建厂→蓝、运营→渐变色
- 客户评价、国家卡片也各自有独立的设计

### 5. 布局 → 更宽松透气
- 内容板块之间 padding 加大
- 卡片之间 gap 从 1rem 增大到 1.2rem
- 浅色背景渐变过渡，避免突然的色块切换
- About section 的 stats 从横向大号改为网格小卡片，更精致
