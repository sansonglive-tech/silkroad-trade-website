# 新丝路·企业主会商业计划书 Word文档生成

## 目标
生成一份完整的企业级商业计划书 Word 文档（.docx），包含目录、多级标题、多列表格、封面页、页眉页脚页码等专业排版。

## 技术路线
最初尝试使用 Node.js `docx` 库，因中文引号字符与 JavaScript 语法冲突（SyntaxError），改为 Python `python-docx` 库。

## 最终方案：Python python-docx
- 脚本：`gen_bizplan.py`（54KB，完整代码）
- 样本：一次性写入所有段落、表格、样式，无需轮询
- 编码：UTF-8，Python 原生处理 Unicode
- 输出文件：`新丝路企业主会商业计划书.docx`（53,824 字节）
